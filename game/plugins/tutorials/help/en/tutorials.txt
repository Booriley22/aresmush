In addition to the help files, Ares offers a series of interactive tutorials to help you get acquainted with the game.

    %xctutorials%xn - Show a list of all available tutorials.
    %xctutorial/start <name>%xn - Start (or restart) a tutorial.
    
You can navigate through the tutorial using these commands:

    %xctutorial%xn - Re-reads the current tutorial page.
    %xctutorial/next%xn - Goes to the next step in the tutorial.
    %xctutorial/prev%xn - Goes to the previous step in the tutorial.
